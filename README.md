# Star Reader

> A kid-friendly reading tracker for Scholastic graded readers. Kids log completed books, earn stars, and spend stars in a parent-managed reward store.

## Stack

- Frontend: Next.js (App Router) + Tailwind CSS
- Backend: Next.js API Routes + NextAuth v5
- Database: PostgreSQL + Drizzle ORM

## Domains

- Production: https://star-reader.johnlawver.com
